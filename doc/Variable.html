Variables

Most programs need to perform a series of calculation before producing output, and thus, need a way to store data temporarily during program execution. In C, as in most programming languages, these storage locations are called variables.

Types

Every variable has a type, which specifies what kind of data it will hold. In other words, the type imparts the characteristics of that variables. It specifies what kind of data may be stored in it and what operations are permissible for that variable. C has a wide variety of types. For now, we will focus on just the following:

  1. int - this is short for integer. Essentially, this type will store a number between -32767 and +32767
  2. float - Essentially these are just decimal point numbers.
  3. char - This is a single character.

Declarations

In order to use a variable in a program, its type must be declared before using the variables. To declare a type you must specify the type of the variable and then give that variable a name. Here are some examples of declaring types:

  • int height;
  • float profit;
  • char c;
  • char c[20]; - This statement creates what is called a "string". A string is just a sequence of characters. This specific declaration says that our string will be a maximum of 20 characters.

Instantiating a Variable

After we have declared the variable, we can then assign it a value. A variable can be assigned any value provided that it is of the same type.

  • Integers are specified as digits. For example: int x = 5;
  • Floats are specified as decimals or just integers. For example: float x = 3.5;
  • Characters are specified using single quotes: char c = 'h';
  • String are specified in double quotes. For example: char s[30] = "hello there";
